# VC 2

## 安装

```bash
npm i vc2
# or
yarn add vc2
```

## 使用

```ts
import { createApp } from 'vue'
import App from './App.vue'
import VC2 from 'vc2'
import 'vc2/lib/index.css'

const app = createApp(App)

app.use(VC2).mount('#app')
```
